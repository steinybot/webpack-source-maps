# webpack-test-app

The goal of this is to have webpack produce a source map which:
1. 

like:
```json
{
  "version": 3,
  "sources": [
    "index.ts"
  ],
  "names": [
    "Error"
  ],
  "mappings": "mBAAA,MAAM,IAAIA,MAAM,U",
  "file": "bundle.js",
  "sourceRoot": "src"
}
```

## Setup

```shell
npm install
```

## Build

```shell
npm run build
```

## Start

```shell
npm start
```
